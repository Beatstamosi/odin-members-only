# TS Template for Express + React App

1. Clone Repo
2. npm install
3. Add Database via pool.js
4. Set up environment variables

.env client:
VITE_API_BASE_URL

.env server:
SESSION_SECRET=
PGHOST=
PGUSER=
PGDATABASE=
PGPASSWORD=
PGPORT=5432 ?

NODE_ENV=development
